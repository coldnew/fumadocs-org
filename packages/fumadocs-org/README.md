# fumadocs-org

Fumadocs integration for Org-mode files.

## Installation

```bash
npm install fumadocs-org
```

## Usage

```typescript
import { createOrgSource } from 'fumadocs-org';

// Basic usage example
const source = createOrgSource();
```

## Development

```bash
# Install dependencies
npm install

# Run tests
npm run test

# Build package
npm run build

# Development mode
npm run dev
```
