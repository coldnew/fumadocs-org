# Plan: Translate Org-mode Drawers to Fumadocs Accordions

## Description

Convert Org-mode drawer blocks to Fumadocs accordion components in MDX output. Drawers are collapsible sections in Org-mode that can be mapped to accordion UI elements.

## Reference

Fumadocs accordion example: https://fumadocs.dev/docs/ui/components/accordion

## Implementation Steps

- Research Org-mode drawer syntax and structure
- Study Fumadocs accordion component API
- Implement drawer-to-accordion conversion in org-mode converter
- Handle nested drawers and content preservation
- Add tests for drawer conversion
