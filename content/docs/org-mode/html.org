#+title: HTML Export in Org-mode
#+description: Exporting Org-mode content with HTML elements

* HTML Export

Org-mode supports embedding raw HTML using =#+HTML:= (or =#+html:=) directives. These directives allow you to include custom HTML elements directly in your Org-mode documents. Both uppercase and lowercase variants work identically.

** Basic HTML Export

You can embed HTML elements using the =#+HTML:= directive:

#+HTML: <div class="alert alert-info">This is an info alert</div>

This HTML will be preserved as-is in the exported output.

** HTML Attributes

HTML attributes are maintained in their original form:

- Standard HTML attributes like =class=, =id=, =style= are preserved
- Custom data attributes are supported
- All HTML5 attributes work as expected

** Example with Attributes

#+HTML: <button class="btn btn-primary" data-toggle="modal">Click me</button>

** Styled HTML

You can include inline styles and complex HTML structures:

#+HTML: <div style="color: red; font-size: 14px; border: 1px solid #ccc; padding: 10px;"><strong>Important:</strong> This is styled text with <em>emphasis</em>.</div>

** HTML in Lists and Tables

HTML can be embedded within other Org-mode elements:

- List item with HTML: #+HTML: <span class="highlight">highlighted text</span>
- Table cell with HTML: | Cell with #+HTML: <code>code</code> |

** Best Practices

- Use semantic HTML elements when possible
- Ensure HTML is valid and well-formed
- Consider accessibility when adding custom HTML
- Test HTML rendering in your target output format
