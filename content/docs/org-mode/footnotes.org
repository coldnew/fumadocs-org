* Footnotes

Org-mode supports footnotes for additional references and citations.

** Syntax Example

#+begin_src text
  Footnotes are supported in Org-mode.[fn:1]

  [fn:1] This is the footnote content.
#+end_src

** Conversion to MDX

Org-mode footnotes are converted to Markdown footnotes, above code will convert to

#+begin_src text
Footnotes are supported in Org-mode.[^1]

[^1]: This is the footnote content.
#+end_src

This allows footnotes to be rendered with appropriate styling in the Fumadocs UI.
