* Mathematical Formulas

Org-mode supports LaTeX-style mathematical expressions for inline and display math.

** Syntax Example

#+begin_src text
Inline math: $E = mc^2$

Display math:
$$\int_0^1 f(x) \, dx$$

Complex formula:
$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$
#+end_src

** Conversion to MDX

Mathematical expressions in Org-mode are preserved and rendered using KaTeX, above code will convert to

#+begin_src text
Inline math: $E = mc^2$

Display math: $$ \int\_0^1 f(x) \\, dx $$

Complex formula: $$ \lim\_{x \to 0} \frac{\sin x}{x} = 1 $$
#+end_src

This allows mathematical expressions to be rendered with appropriate styling in the Fumadocs UI.
