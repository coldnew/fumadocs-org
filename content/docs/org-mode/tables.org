* Tables

Org-mode supports simple and complex tables with automatic formatting.

** Syntax Example

#+begin_src text
| Name    | Age | Occupation  |
|---------+-----+-------------|
| Alice   |  25 | Engineer    |
| Bob     |  30 | Designer    |
| Charlie |  35 | Manager     |

| Item | Value |
|------|-------|
| A    | 1     |
| B    | 2     |
#+end_src

** Conversion to MDX

Org-mode tables are converted to Markdown tables, above code will convert to

#+begin_src text
## Simple Table

| Name    | Age | Occupation |
| ------- | --- | ---------- |
| Alice   | 25  | Engineer   |
| Bob     | 30  | Designer   |
| Charlie | 35  | Manager    |

## Table

| Item | Value |
|------|-------|
| A    | 1     |
| B    | 2     |
#+end_src

This allows tables to be rendered with appropriate styling in the Fumadocs UI.
